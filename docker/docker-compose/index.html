简介

docker compose 是用来配置和运行多容器服务的工具，通过 docker-compose 命令可轻松对多个容器进行如下操作：

（1）启动，停止和重建服务

（2）查看正在运行的服务的状态

（3）通过流输出正在运行的服务的日志

（4）对某个服务执行命令
