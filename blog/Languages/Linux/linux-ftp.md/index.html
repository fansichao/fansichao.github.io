注意事项：

  • 配置单向网关或防火墙策略时，外部网络变动时，容器内部不会变动，容器以及 docker 都需要重启才可以更新。

问题记录

500 OOPS:cannot change directory:/root

ftplib 连接时 selinux 未关闭导致的这个错误。

解决方法

  1. 关闭 selinux
  2. setsebool ftpd_disable_trans 1 或者 setsebool ftp_home_dir 1
