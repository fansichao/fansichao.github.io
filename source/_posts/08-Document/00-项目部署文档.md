### 关闭 selinux

```bash
# 关闭selinux ，selinux不关闭，会容易导致各种情况不允许导致安装配置等报错。
sed -i "s/SELINUX=enforcing/SELINUX=disabled/" /etc/selinux/config
# 关闭selinux防火墙
setenforce 0
```

### 关闭 防火墙

```bash
# 关闭防火墙
systemctl disable firewalld
systemctl stop firewalld
```

```bash

docker exec -it fdm_graph /bin/bash
# 创建所需目录
mkdir -p /data/hugegraph/rocksdb_data
# 目录授权
chmod 777 -R /data/hugegraph

# 切换用户
su - fdm
# 切换目录
cd /home/fdm/hugegraph/hugegraph-0.9.2
# 初始化数据库
bin/init-store.sh
```

```bash
# 容器内部
cp -r jdk1.8.0_171 /data/.
```

## 问题记录

### 系统环境配置问题

#### Cannot update read-only repo

参考链接: [Cannot update read-only repo](https://blog.csdn.net/hm0406120201/article/details/104553205)

```bash
[root@localhost ~]# python
bash: python: command not found...
Failed to search for file: Cannot update read-only repo
```

解决方案

- dnf install -y centos-release-stream
- dnf update
- 重启 CentOS8
