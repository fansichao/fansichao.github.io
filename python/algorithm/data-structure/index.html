算法

迭代法

关键点:

  • 将一个原问题修改为子问题
  • 对子问题重复调用计算
  • 特性：
    • 重复计算多，耗时高，时间复杂度高

动态规划

  • 关键点:
    • 将一个原问题修改为子问题
    • 试图只解决每个子问题一次，一旦其问题已经解决，会将其存储，从而避免重复计算
    • 特性:
      • 尽量避免重复计算，时间复杂度低。
  • 关键概念
    • 最优子结构
    • 边界
    • 状态转移公式
